All classes include a baseline and a sample recording, lasting for 120 and 180 seconds respectively.

Everything was recorded using the following heating profiles:
S1, S2 - "[Sequential] 100,200,320 C / 150 ms each, 250 ms ODR sleep"
S3, S4 - "[Sequential] Bosch 10-step: 320,100x3,200x3,320x3 C / 150 ms each, 500 ms ODR sleep"
S5, S6 - "[Parallel] Bosch standard 10-step: 320,100,100,100,200,200,200,320,320,320 C"
S7, S8 - "[Parallel] Low-temp      100 -> 250 C  (10 steps)"

(Parallel mode is just like sequential, except that the sensors are cycled faster.)
